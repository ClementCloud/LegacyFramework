# The Sunshine Trust - Interactive Guide

An interactive, guided tour through The Sunshine Trust framework - a multi-generational family wealth management system.

## 🌟 Features

- **Step-by-step guided tour** through all 8 components of the Trust
- **Interactive navigation** with progress tracking
- **Beautiful visual design** with gradients and animations
- **Mobile responsive** - works on all devices
- **Single-page app** - no dependencies needed

## 📖 What's Covered

1. **The Foundation** - Protected principal and yield distribution
2. **Governance Structure** - 3-member Trustee Board
3. **Money Allocation** - 60-20-10-10 split explained
4. **Five Tiers** - Performance-based distribution levels
5. **Scoring System** - How beneficiaries are evaluated
6. **Four Pillars** - Distributions, Family Bank, Insurance, Summit
7. **Insurance Multiplier** - Optional capital flywheel program
8. **Core Values** - Cultural foundation of the Trust
