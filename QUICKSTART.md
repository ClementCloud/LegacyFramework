# 🚀 QUICK START - 3 Easy Steps

## Option 1: Super Easy (No Git Required)

### Step 1: Create GitHub Repository
1. Go to https://github.com/new
2. Repository name: `sunshine-trust-guide`
3. Privacy: **Private** ✅
4. Click **Create repository**

### Step 2: Upload File
1. Click **uploading an existing file**
2. Drag `index.html` into the upload area
3. Click **Commit changes**

### Step 3: Enable GitHub Pages
1. Click **Settings** tab
2. Click **Pages** in sidebar
3. Under Source, select: **main** branch, **/ (root)** folder
4. Click **Save**
5. Wait 2-3 minutes

### Your Site is Live! 🎉
Visit: `https://YOUR-USERNAME.github.io/sunshine-trust-guide/`

---

## Option 2: Using Git (For Developers)

```bash
# 1. Navigate to the folder
cd sunshine-trust-pages

# 2. Initialize and commit
git init
git add .
git commit -m "Initial commit"

# 3. Create repository on GitHub
# Go to https://github.com/new
# Name it: sunshine-trust-guide

# 4. Push to GitHub (replace YOUR-USERNAME)
git remote add origin https://github.com/YOUR-USERNAME/sunshine-trust-guide.git
git branch -M main
git push -u origin main

# 5. Enable Pages in repository Settings → Pages
```
